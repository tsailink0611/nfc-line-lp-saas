# NFC営業支援SaaS — 実践ロードマップ

## 全体の流れ

```
Phase 0（今週）    自分で体験する
Phase 1（2-4週後）  デモキット完成
Phase 2（2-3ヶ月後） 知り合いに見せる
Phase 3（4-6ヶ月後） テスト導入1社目
Phase 4（8ヶ月後）  退職→本格始動
```

---

## Phase 0: 自分で体験する（今週〜2週間）

1. **NFCカードを購入**（Amazon: NTAG215 x 5枚、約1,500円）
2. **自分のLPを管理画面から登録**
3. **NFCカードにURL書き込み**（iOS: NFC Tools / Android: NFC TagWriter）
   - 書き込むURL: `https://nfc-line-lp-saas.tsailink0611.workers.dev/n/{token}`
   - tokenは管理画面 > NFC管理 > 発行で取得
4. **自分でタッチして体験する**
5. **気づいた改善点をメモする**

## Phase 1: デモキット完成（2-4週間後）

1. 自分のLPを本気で作り込む
2. デモ用トークスクリプトを準備
   - 「ちょっと触ってみてもらえますか？」→ NFCカードを渡す
   - 「これ、御社の営業全員に持たせたらどうなると思います？」

## Phase 2: 知り合いに見せる（2-3ヶ月後）

ターゲット順:
1. 一番仲のいい経営者1人（練習台）
2. 不動産・建築系の知り合い
3. 酒屋・ビールメーカーの知り合い

見せ方: 説明しない。まずタッチさせる。

## Phase 3: テスト導入1社目（4-6ヶ月後）

1. ヒアリング（2h）
2. セットアップ（3h）
3. NFCカード設定・配布（2h）
4. 営業員向け説明会（1h）

## Phase 4: 退職→本格始動（8ヶ月後）

- 初月3社、3ヶ月で10社が目標

---

## 開発の優先順位

| 優先度 | 機能 | いつ作るか |
|--------|------|----------|
| 今すぐ | カスタムドメイン設定 | Phase 0 |
| 体験後 | 使い勝手の微調整 | Phase 0-1 |
| 1社目導入前 | company_adminロール | Phase 3直前 |
| 導入後 | アクセス解析 | Phase 3 |
| 10社到達後 | スーパー管理者画面 | Phase 4以降 |

**原則: 使ってみて「これがないと困る」と実感したものだけ作る。**
